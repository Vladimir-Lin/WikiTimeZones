HtPassword - a MediaWiki:Extension for accessing Apache .htpasswd file
 
Author : Vladimir Lin aka Foxman <lin.vladimir@gmail.com>

Project URL : https://vladimir-lin.github.io/HtPassword/
GIT : https://github.com/Vladimir-Lin/HtPassword

This extension is so simple that I have no need to write a whole bunch documents.  This page is quite enough.

Download

Goto your MediaWiki installation directory, such as somewhere/MediaWiki/ , enter extensions directory, and then give the following command:

git clone https://github.com/Vladimir-Lin/HtPassword

This will create a directory named "HtPassword"

Installation

In somewhere/MediaWiki/extensions/HtPassword directory, find a file called config.php, open it in a code editor, and modify:

$wgHtPassword = "D:\\Programs\\XAMPP\\apache\\bin\\htpasswd.exe" ;

In somewhere/MediaWiki directory, find LocalSettings.php, open it up, and add the following line:

add wfLoadExtension ( 'HtPassword' ) ;

Usage

Syntax: {{ #HtPassword: htpasswd_file | language | width | Yes | canDelete }}

Example :
{{ #HtPassword: /htpasswd | zh-TW | 25% | Yes | Yes }}
